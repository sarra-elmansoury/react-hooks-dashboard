# Smart Product Dashboard

## Description
A smart product management dashboard built with React. It implements a fully functional global state utilizing `useReducer` and `useContext`, performance optimization using `useMemo` and `useCallback`, and clean abstraction via custom hooks. The UI mimics modern application dashboards with a clean layout and responsive design.

## Features Added
- Add Product
- Delete Product
- Toggle Stock
- Search by Name
- Filter by Stock Status
- Fake API Loading State
- Pagination
- Dark Mode
- LocalStorage Data Persistence
- React Bootstrap Integration

## Installation

2. `npm install react-bootstrap bootstrap`
